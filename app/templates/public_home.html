Platform Features

Everything you need for comprehensive ransomware defense training

Simulation Scenarios

Pre-built ransomware scenarios covering crypto-lockers, wipers, exfiltration, and multi-stage APT attacks. Custom scenario builder included.

Agent-Based Execution

Deploy Windows agents to simulate real ransomware behavior: file encryption, VSS deletion, persistence, lateral movement.

ELK SIEM Integration

Connect to Elasticsearch for real-time alert correlation. Auto-refresh dashboards, MITRE ATT&CK heatmaps, and alert feeds.

Live Alerts Feed

Real-time alert streaming from Elastic Security. Filter by severity, host, and MITRE technique.

SOAR Auto Response

Lab-safe automated response playbooks: host isolation, process termination, user disablement, file quarantine.

MITRE ATT&CK Reports

Automatic technique mapping, behavior profiling, compliance reports (GDPR, HIPAA), and forensic timelines.
